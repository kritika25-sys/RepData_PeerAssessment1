y<-0
a<-NULL
a1<-nrow(subset(x,x$date==unique(x$date)[1]))
for (i in 1:a1) {
  j<-i

There are 2304 missing values in the dataset.

  • A strategy for filling in the missing values in the dataset. We will be filling in the missing values for a particular day in the dataset with the mean of the number of steps taken in that day.

  • A new dataset that is equal to the original dataset but with the missing data filled in

y<-NULL
x1<-x
for (i in 1:length(unique(x1$date))) {
  d1<-subset(x1, x1$date==unique(x1$date)[i])
  y[i]<-mean(d1$steps, na.rm = TRUE)
}
m<-which(is.nan(y)==TRUE)
for (i in m) {
  y[i]<-0
}
i<-1
j<-1
while (i<=nrow(x1)) { 
    for (k in 1:a1) {
      if(is.na(x1$steps[k+i-1])==TRUE)
        x1$steps[k+i-1]<-y[j]
    }
  j<-j+1
   i<-i+a1
